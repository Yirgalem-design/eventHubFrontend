Technology Stack & Tools – EventHub Planner
1. Frontend
Technologies

HTML5 – Structure of the webpage

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Handles UI logic, API requests, event rendering, and CRUD functionality

Tools

Bootstrap 5 – Responsive design, grid system, buttons, form styling

Fetch API – Communicates with the backend using REST endpoints

Vercel (Deployment) – Hosts the frontend as a static site

2. Backend
Technologies

Node.js – JavaScript runtime environment

Express.js – Web server framework used to build REST APIs

PostgreSQL – Relational database for storing events

Tools

PG (node-postgres) – PostgreSQL client library used to connect to the DB

dotenv – Loads environment variables from a .env file

Render (Deployment) – Hosts the backend as a web service

GitHub – Version control and source code repository

3. Database
Technologies

PostgreSQL – Stores all event data (title, description, date, time, location)

Tools

Render PostgreSQL – Cloud-hosted managed database

SQL – Used to create and manage database tables

4. Development Tools

Visual Studio Code – Source-code editor

Git – Version control

Browser Developer Tools – Debugging network requests and UI

Summary

EventHub Planner uses a full-stack JavaScript ecosystem:

Frontend: HTML, CSS, Bootstrap, JavaScript

Backend: Node.js, Express.js

Database: PostgreSQL

Deployment: Vercel + Render

Tools: GitHub, dotenv, pg, VSCode
